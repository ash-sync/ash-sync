## ⚔️ Challenges I'm Tackling

- 📐 **Mastering System Design** — learning how to think at scale: load balancing, caching strategies, database sharding, and distributed systems
- 🚀 **Performance Optimization** — reducing bundle size, improving API response times, minimizing re-renders, and writing efficient MongoDB/PostgreSQL queries
- 🔄 **State Architecture** — structuring Redux Toolkit + RTK Query in large apps without creating spaghetti logic
- 🐳 **DevOps & Deployment Pipelines** — containerizing apps with Docker and managing cloud deployments on AWS
